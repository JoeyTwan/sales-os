# Sales OS 产品规划

## 愿景

打造个人销售的大脑。

## 原则

1. AI永远不直接修改数据
2. 用户拥有最终确认权
3. AI负责建议
4. 用户负责决策
5. 简洁优先于复杂

## 模块设计

### 1. 工作台

**功能**：

快速记录所有想法

### 2. 收件箱

**作用**：

所有输入先进入收件箱

**状态**：

- PENDING - 待处理
- CONFIRMED - 已确认
- ARCHIVED - 已归档

### 3. 客户管理

- 客户信息
- 客户等级
- 客户状态
- 下一步动作
- 跟进记录

### 4. 任务管理

- 今日
- 本周
- 未来

**视图**：

- 列表视图
- 日历视图

### 5. 知识库

- NotebookLM入口

**原则**：不在Sales OS内部构建RAG

### 6. 战略大脑

**读取**：

- 客户
- 任务
- 跟进记录

**输出**：

- 风险分析
- 机会分析
- 下一步建议

## AI原则

### 禁止

- AI自动创建客户
- AI自动修改客户
- AI自动删除数据

### 允许

- AI提出建议
- AI生成摘要
- AI生成战略分析

**核心规则**：所有操作必须用户确认
